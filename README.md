# Анализ зарплат программистов в Москве (HeadHunter + SuperJob)

Скрипт собирает вакансии программистов с сайтов HeadHunter и SuperJob, анализирует зарплаты и выводит статистику в виде таблицы в консоли.

---

## Что делает скрипт

Программа автоматически:

- обращается к API HeadHunter и SuperJob
- ищет вакансии программистов по популярным языкам
- фильтрует вакансии по городу (Москва)
- рассчитывает:
  - общее количество вакансий
  - количество вакансий с указанной зарплатой
  - среднюю зарплату
- выводит результат в удобной таблице

---

## Как считается зарплата

В вакансиях зарплата указывается в виде диапазона:

- от (`salary_from`)
- до (`salary_to`)

Логика расчёта:

- если есть оба значения → берётся среднее
- если есть только одно → используется оно
- если зарплата не указана → вакансия не учитывается

### HeadHunter
Используются поля:
- salary.from
- salary.to
- salary.currency

Учитываются только вакансии в рублях (RUR).

### SuperJob
Используются поля:
- payment_from
- payment_to
- currency

Если значения равны 0, считается, что зарплата не указана.

## Настройка API-ключа SuperJob

Для работы с SuperJob необходим API-ключ.

1. Получите ключ: https://api.superjob.ru/

2. Создайте файл .env:

SUPERJOB_API_KEY=ваш_ключ

Важно:
- без кавычек
- без пробелов
- файл должен называться именно .env

3. Добавьте .env в .gitignore:

.env
__pycache__/

## Возможные ошибки

ValueError: Нет SUPERJOB_API_KEY в переменных окружения

Проверь:
- файл .env существует
- имя переменной написано правильно
- файл находится рядом со скриптом
